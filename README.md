# 📱💳 App Prepago Móvil

Aplicación móvil diseñada para la **recarga y gestión de saldo telefónico prepago**, desarrollada con tecnologías web modernas enfocadas en rendimiento y experiencia de usuario.

---

## 🚀 Descripción

**App Prepago Móvil** permite a los usuarios recargar saldo, consultar su consumo y administrar su línea telefónica de forma rápida y segura desde su dispositivo móvil.

La aplicación está construida con **TypeScript, HTML y CSS**, ofreciendo una interfaz ligera, responsive y fácil de usar.

---

## 🎯 Objetivos

- Facilitar recargas de saldo de manera rápida  
- Permitir consulta de consumo y saldo  
- Mejorar la experiencia del usuario móvil  
- Ofrecer una interfaz clara y accesible  

---

## ✨ Características

### 💳 Recargas
- Recarga de saldo en tiempo real  
- Selección de montos  
- Confirmación de transacciones  

### 📊 Consulta de Saldo
- Visualización de saldo actual  
- Historial de recargas  

### 📱 Interfaz Móvil
- Diseño responsive  
- Navegación intuitiva  
- Optimización para dispositivos móviles  

### 🔐 Seguridad
- Validación de datos  
- Protección de información del usuario
